July,2020

Online Educational Application With Intelligent Analytics In Android

As a part of my internship at a Noida based Startup - Classplus, I worked on a B2B application catering to ... all the needs of an educational institution. The application is an ecosystem of Tutors, Students and Parents of Coaching Centers across India. The feature ranges from Institute Management like Student Attendance, Enquiries, Payments, HomeWorks etc. The new modules I got a chance to work on were : Home & Store Revamping using Dynamic Cards and Coupon(Referral) Management System along with analytics using Clevertap and Servicate. These features improved the overall user experience of students and tutors while maintaining the quality of the builds with respect to memory, network and battery optimizations. All the features and coding used latest MVP Architecture,were coded in Kotlin and backend was developed using GraphiQL. The application is purchased by tutors who use intelligent systems to cater to the needs like batch creation, management ,online videos, pdfs, live classes, selling courses, attendance, scheduling tests and releasing analytics related to events, broadcast messages, take feedback, share resources analyze results by data visualization, track clicks of activities and log events for improving content and a wide varied applications.
I was the first intern to get felicitated with a "Rockstar Rookie" badge in recognition of my exceptional performance in this project.

October,2018

Face Filter |Computer Vision

Face Recognition Project aims at identifying facial features for detection. ... The project was implemented using Python and openCV,the accuracy was achieved upto 68%. The Haar Cascade and LBPH Algorithm was used.However after proper study and research another algorithm using "Face Net" by Google was identified. We propose a framework which classifies the local facial regions of both visible and thermal face images into biometric (regions without disguise) and non-biometric (regions with disguise) classes. The biometric patches are then used for facial feature extraction and matching. The performance of the algorithm is evaluated on the IHTD In and Beyond Visible Spectrum Disguise database that is prepared by the authors and contains images pertaining to 75 subjects with different kinds of disguise variations. The experimental results suggest that the proposed framework improves the performance compared to existing algorithms, however there is a need for more research to address this important covariate. This can increase accuracy to more than 90%.
